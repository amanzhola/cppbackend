## 💡 Почему обычный Windows здесь не подходит

| Вопрос                                                     | Полное описание                                                                                                                                  | Итог                                         |
| ---------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------- |
| Почему нельзя просто делать всё в Windows CMD / PowerShell | В курсе для локальной проверки используются Linux shell-скрипты вида `./prepare_test.sh` и `./sum/run.sh`                                        | Обычный Windows shell для этого не подходит  |
| Что именно не подходит                                     | Windows CMD / PowerShell подходят для Git, работы с файлами и обычных Windows-команд, но не подходят как основная среда для `.sh`-скриптов курса | Для этой задачи нужен Linux shell            |
| Что нужно вместо этого                                     | Либо платная удалённая Linux-среда, например Yandex Cloud VPS, либо бесплатная локальная Linux-среда через WSL                                   | Для этой задачи был выбран WSL               |
| Почему WSL лучше                                           | WSL позволяет бесплатно получить Linux-окружение прямо на своём компьютере и запускать `.sh`-скрипты так, как ожидает курс                       | WSL — основная рабочая среда для этой задачи |

## 🐧 Почему здесь используется WSL

| Причина                | Полное описание                                                                                                  | Итог                                 |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------- | ------------------------------------ |
| Бесплатная Linux-среда | Вместо постоянной работы через платный сервер можно использовать WSL Ubuntu локально на своём компьютере         | Это удобнее и дешевле                |
| Совместимость с курсом | Скрипты курса и тестовый фреймворк ожидают Linux / bash-среду                                                    | WSL решает эту проблему              |
| Удобство               | Можно одновременно работать с Linux-путями и открывать файлы через Windows Explorer                              | Среда становится удобной и привычной |
| Практический смысл     | Для этой задачи настройка среды важнее самого решения, потому что именно она позволяет проходить локальные тесты | WSL — главный результат этой задачи  |
