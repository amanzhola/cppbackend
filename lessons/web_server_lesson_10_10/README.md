# 🚀 Практическое занятие №3

## Проверить HTML-ответ

---

# 📌 Кратко

| 📌 Что это                         | 📖 Полное описание                                                                                                                                                                                                                  |
| ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Учебный HTTP-сервер с HTML-ответом | В этом задании сервер принимает HTTP-запрос, выводит его структуру, формирует HTML-ответ `<strong>Hello</strong>` и отправляет его клиенту с правильным заголовком `Content-Type`, благодаря чему браузер отображает текст как HTML |

---

# 🎯 Цель

| 🎯 Что нужно понять           | 📖 Что это означает на практике                                     |
| ----------------------------- | ------------------------------------------------------------------- |
| HTML зависит от Content-Type  | Браузер рендерит HTML только если сервер явно указывает `text/html` |
| Теги влияют на отображение    | Тег `<strong>` делает текст жирным                                  |
| Сервер управляет отображением | Именно backend решает, как клиент будет интерпретировать ответ      |

---

# ⚙️ Основной код

```cpp
StringResponse res{http::status::ok, req.version()};
res.set(http::field::server, "Lesson10_10");
res.set(http::field::content_type, "text/html; charset=utf-8");
res.keep_alive(false);
res.body() = "<strong>Hello</strong>\n";
res.prepare_payload();
```

---

# 🧠 Главная идея

| 🧠 Ключевая логика                                                  | 📖 Смысл                                                                                                          |
| ------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| `Content-Type: text/html + HTML в body → браузер рендерит страницу` | Сервер отправляет HTML и сообщает браузеру, что это HTML, поэтому текст отображается как форматированная страница |
