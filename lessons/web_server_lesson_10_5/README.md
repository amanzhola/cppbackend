# 🚀 Практическое занятие №3

## Обработка разных HTTP-маршрутов

---

# 📌 Кратко

| 📌 Что это              | 📌 Описание                                                                                                            |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Сервер с маршрутизацией | Сервер анализирует `req.target()` и возвращает разные ответы и статус-коды в зависимости от пути и метода HTTP-запроса |

---

# 🎯 Цель

| 🎯 Что нужно понять | 📖 Что это означает |
| ------------------- | ------------------- |
| Разные маршруты     | Сервер понимает URL |
| Статус-коды         | 200, 404, 405       |
| Метод запроса       | GET vs POST         |

---

# 🧠 Идея

| 🧠 Логика      | 📖 Смысл              |
| -------------- | --------------------- |
| `req.target()` | определяет маршрут    |
| `req.method()` | определяет HTTP-метод |
| MakeResponse   | возвращает ответ      |

---

# 🌐 Маршруты

| 🌐 Запрос         | 📥 Ответ                            |
| ----------------- | ----------------------------------- |
| `GET /`           | `Main page`                         |
| `GET /hello`      | `Hello!`                            |
| `GET /time`       | `This could be current server time` |
| любой другой путь | `404 Not Found`                     |
| не GET            | `405 Method Not Allowed`            |

---
