# 🚀 Практическое занятие №5

## Разные ответы в зависимости от пути

---

# 📌 Кратко

| 📌 Что это                           | 📖 Полное описание                                                                                                                             |
| ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Учебный HTTP-сервер с маршрутизацией | В этом задании сервер анализирует путь запроса и возвращает разные ответы в зависимости от `target`, включая разные HTML-страницы и ошибку 404 |

---

# 🎯 Цель

| 🎯 Что нужно понять   | 📖 Что это означает на практике               |
| --------------------- | --------------------------------------------- |
| Анализ request target | Сервер определяет, какой путь запросил клиент |
| Выбор ответа          | Для каждого пути формируется свой ответ       |
| Управление статусом   | Сервер может вернуть 200 или 404              |
| Выбор Content-Type    | HTML и текст возвращаются с разными типами    |

---

# ⚙️ Основной код

```cpp
if (req.target() == "/") {
    res.set(http::field::content_type, "text/html; charset=utf-8");
    res.body() = "<h1>Main page</h1>\n";
} else if (req.target() == "/hello") {
    res.set(http::field::content_type, "text/html; charset=utf-8");
    res.body() = "<strong>Hello</strong>\n";
} else {
    res.result(http::status::not_found);
    res.set(http::field::content_type, "text/plain; charset=utf-8");
    res.body() = "404 Not Found\n";
}
```

---

# 🧠 Главная идея

| 🧠 Ключевая логика                                    | 📖 Смысл                                                      |
| ----------------------------------------------------- | ------------------------------------------------------------- |
| `req.target() → выбор ответа → формирование response` | Сервер анализирует путь запроса и возвращает разный результат |
